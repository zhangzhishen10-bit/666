## 科研实践项目介绍
# 自动化视觉模型方案
# 1. 任务概述本系统旨在通过工业相机实时捕捉自动化实验台面（Deck）图像，利用深度学习模型实现从宏观环境识别到微观位点判断的闭环控制。其核心挑战在于：高密度的枪头识别、透明材质的反光干扰，以及像素坐标向机械臂物理坐标的高精度转换。
# 2. 视觉算法架构（重点）本方案采用 “全局检测 + 局部精细化感知” 的双级联动架构。
2.1 第一级：基于 YOLO 的全局布局检测（Global Context）模型选型：采用 YOLOv11-m。该模型在保持推理实时性的前提下，对重叠物体具有极强的特征提取能力。任务目标：区域分割：将台面图像划分为 96-well-plate (96孔板)、tip-rack (枪头盒)、waste-bin (废液盒) 等 ROI（感兴趣区域）。状态初判：识别各区域是否被占用、耗材类型是否与协议一致。输出逻辑：一个包含bbox的横纵坐标、长宽、置信度、标签的字典YOLO 输出枪头盒的精确坐标，作为下一级模型的输入触发信号。
2.2 第二级：基于 SAM 3 的枪头位点细分（Micro-Perception）在获取枪头盒区域图像后，单纯依靠目标检测难以判断“某个特定孔位是否有枪头”（尤其是透明枪头）。此时引入 SAM 3 (Segment Anything Model 3)。Prompt 引导机制：将 YOLO 输出的字典裁切区域输入 SAM 3。利用 SAM 3 的 网格提示（Grid Prompt） 功能，在裁切后的枪头盒区域内生成覆盖所有潜在位点的采样点。像素级分割与逻辑判定：SAM 3 对每个位点进行掩码（Mask）生成。空/满判定逻辑：通过计算每个掩码区域内的颜色矩（Color Moment）或纹理复杂度（Texture Entropy）。若该位点掩码内的特征值与背景（枪头架底座）一致，判定为 Empty。若检测到明显的塑料边缘或光折射特征，判定为 Occupied。几何中心锁定：利用 SAM 3 生成的高精度掩码，通过二值化处理后的重心计算公式获取枪头顶端的精确像素中心。
# 3. 问题：需要编写代码把SAM3精细提取的枪头盒的枪头信息（空和有的位置）转化为机器代码中设置的plate的信息，让机器根据这个信息来精确抓取。同时我们可能也需要一些训练用的数据集用来模型来训练yolo模型。
